## Deskripsi Proyek

Washly Go Laundry merupakan aplikasi E-Laundry yang dirancang untuk membantu  
proses pengelolaan layanan laundry, mulai dari pemesanan, pembayaran,  
hingga monitoring status pesanan secara terkomputerisasi.

Dokumentasi ini digunakan sebagai pendukung analisis, perancangan,  
dan presentasi proyek.

---

## Isi Repository

### 1. All Diagram
Berisi seluruh diagram UML yang digunakan dalam proyek, antara lain:
- Use Case Diagram  
- Activity Diagram  
- Sequence Diagram  
- Class Diagram  

### 2. Dokumen RPL
Berisi dokumen perancangan sistem, seperti:
- Software Requirements Specification (SRS)  
- Software Design Description (SDD)  
- Software Test Description (STD)  

### 3. Washly Go.pptx
File presentasi proyek **Washly Go Laundry**  
yang digunakan untuk pemaparan hasil analisis dan perancangan sistem.

### 4. Figma Link
Berisi link desain antarmuka (UI/Mockup) aplikasi  
yang dibuat menggunakan Figma.
